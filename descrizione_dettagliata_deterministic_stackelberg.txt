DESCRIZIONE DETTAGLIATA - deterministic_stackelberg.py
=======================================================

STATO ATTUALE DEL CODICE
========================

Il file implementa un modello SIR deterministico con Stackelberg game:
- leader: governo, che sceglie il controllo c_s;
- follower: cittadini, che scelgono la socialita x* in base a prescrizione e rischio;
- dinamica epidemiologica: compartimenti S, I, H, R con ospedalizzazione ritardata.

La versione corrente esegue nella run principale:
1) simulazione con best response quadratica
2) simulazione con best response logaritmica
3) confronto finale tra i due esiti
4) salvataggio di 4 grafici PNG

Le funzioni di scansione e calibrazione sono presenti nel file ma NON vengono
eseguite automaticamente nel flusso principale.


MODELLO (FORME USATE NEL CODICE)
================================

1) Prescrizione sociale dal controllo del governo
-------------------------------------------------
x_bar(c_s) = 1 / (1 + kappa * c_s)   [tipo default: logistica]

2) Rischio percepito
--------------------
p_t = clip(rho_rischio * (I_t + H_t) / N, 0, 1)

Nella simulazione giornaliera si usa I_t + H_t per il rischio percepito.

3) Best response cittadini
--------------------------
3.1 Quadratica (chiusa)
x*_t = clip(x_bar - (lambda_rischio * p_t) / eta_compliance, 0, 1)

3.2 Logaritmica (numerica su griglia)
U(x) = a * log(x / x_bar + epsilon) - lambda_rischio * p_t * x

4) Fattore di contatto sociale
------------------------------
new_infections_t = f(x*_t) * beta * S_t * I_t / N
f(x) = x^potenza, con default potenza = 1.0

5) Compartimenti
----------------
S_{t+1} = S_t - new_infections_t
I_{t+1} = I_t + new_infections_t - new_hospitalizations_t - gamma * I_t
H_{t+1} = H_t + new_hospitalizations_t - gamma_h * H_t
R_{t+1} = R_t + gamma * I_t + gamma_h * H_t

con:
- new_hospitalizations_t = h * new_infections_{t-tau} (dopo ritardo tau)
- gamma_h = 1 / degenza_media_ospedaliera

6) Costo del governo su finestra MPC
------------------------------------
J(c_s) = J_epidemico + J_controllo

J_epidemico:
somma nel tempo di
C_t = c_i * I_t + k * (exp(alpha * (H_t / capacita_ospedaliera)) - 1)

J_controllo:
lambda_reg_controllo * c_s^2 * (orizzonte + 1)

FLUSSO RUN PRINCIPALE (EFFETTIVO)
=================================

Il blocco finale del file usa guardia if __name__ == "__main__" e richiama main(), che esegue:

1) stampa intestazione simulazione
2) run completa quadratica con simula_sir_stackelberg_con_controllo_periodico(..., tipo_best_response="quadratica")
3) run completa logaritmica con simula_sir_stackelberg_con_controllo_periodico(..., tipo_best_response="logaritmica")
4) calcolo metriche:
   - picco infetti e giorno picco
   - picco ospedalizzati H e giorno picco H
   - costo epidemico totale con controllo variabile
   - compliance media x*/x_bar
   - gap medio x* - x_bar
5) stampa dei delta (logaritmica - quadratica)
6) plotting e salvataggio output PNG
7) mostra grafici se mostra_grafici_default = True
